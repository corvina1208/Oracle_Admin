**■ 데이터베이스를 구성하는 파일 8가지**  
**1. data file :** data 를 저장하는 파일  
**2. redo log file :** 변경 사항을 기록하는 파일  
**3. control file :** DB 의 구조 정보를 담고있는 파일
----- 위 3 개는 하나라도 손상되면 DB가 shutdown 된다 -----
