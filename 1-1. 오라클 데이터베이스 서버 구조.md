■ 실습

#1. SYS 유저로 접속

$ sqlplus / as sysdba

#2. 오라클 DB 의 이름 확인

SYS > select name from v$database;

#3. 오라클 메모리(instance) 의 이름 확인

SYS > select instance_name from v$instance; 



문제1. 오라클 메모리가 잘 열려있는지 확인하기

SYS > select instance_name, status
         from v$instance;
