**1. Database** : 가장 큰 저장 단위  
오라클을 설치하면 데이터베이스를 가장 먼저 생성한다.  
※ DB 이름 확인
```
select name from v$database;
```
<br/>
**2. Tablespace** : 하나의 Database 는 여러 개의 Tablespace 로 구성되어 있다.  
※ Tablespace 확인
```
select tablespace_name, file_name from dba_data_files;
```
