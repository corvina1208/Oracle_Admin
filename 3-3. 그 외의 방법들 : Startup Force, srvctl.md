**▣ 그 외의 방법들 : Startup Force, srvctl**  

<br/>
<br/>
<br/>
  
Shutdown 옵션 외에 DB 를 시작&종료할 수 있는 방법 2가지  

<br/>
<br/>
<br/>

**1. STARTUP FORCE**  
```
startup force
```  
shutdown abort + startup 를 사용한 것과 동일하다.  

<br/>
<br/>

**2. 클러스터 명령어 이용**  
```
$ srvctl stop database -d [인스턴스명] -o abort
$ srvctl start database -d [인스턴스명]

$ crs_stop -all
$ crs_start -all
```
